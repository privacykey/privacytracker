# Privacy categories

Fourteen data categories taken straight from Apple's App Store privacy taxonomy.
Each entry has an emoji glyph, a colour token (mapped to `--blue/--red/--orange/--text-3`),
a label and a description. Defined in `lib/privacy-meta.ts` → `CATEGORY_META`.
